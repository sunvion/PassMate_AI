# PassMate_AI
PassMate AI는 다양한 시험을 준비하는 수험생을 위한 AI 기반 학습 지원 플랫폼이다.

기존 문제은행 서비스는 문제 제공 및 채점 기능에 집중되어 있어 사용자가 자신의 약점을 분석하고 학습 방향을 설정하기 어렵다.

본 프로젝트는 시행처가 공개한 문제은행 및 기출문제를 기반으로 사용자의 학습 이력을 분석하여 다음 기능을 제공한다.

- 취약 단원 분석
- 오답 복습 관리
- AI 기반 문제 해설
- 개인 맞춤형 문제 추천
- 학습 성과 시각화

**또한 사용자는 별도의 검색이나 외부 AI 서비스를 이용하지 않고도 문제 풀이 과정에서 발생한 궁금증을 플랫폼 내에서 해결할 수 있다.**


 사용자가 틀린 문제에 대해 추가 질문을 입력하면,

 해당 문제와 연계된 AI 해설 및 개념 설명을 즉시 제공
